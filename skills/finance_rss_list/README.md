# Feedspot Financial RSS List

Source page:
- https://rss.feedspot.com/financial_news_rss_feeds/

Extraction notes:
- Extracted from Feedspot page content on 2026-02-14.
- The page title says "50 Best Financial News RSS Feeds", but the visible indexed entries are 49 (`1` to `49`) in the rendered content.
- This folder contains:
  - `feedspot_financial_rss_urls.txt`: validated full RSS URL index.
  - `feedspot_financial_rss_catalog.csv`: full visible entries with website URL and extraction status.
  - `categories/*.txt`: categorized feed lists for quota-based selection.
  - `category_index.json`: canonical URL -> category mapping.
  - `CATEGORIZATION.md`: category definition and assignment rules.
  - `additional_rss_validated_ok.json` / `additional_rss_validated_bad.json`: validation reports for expanded candidates.
